# Trigger Bot Frontend

Веб-интерфейс администратора для Trigger Bot.

## Стек

- React 18 + TypeScript
- Vite
- TailwindCSS

## Разработка

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

Результат сборки будет в папке `dist/`.

## Переменные окружения

Скопируйте `.env.example` в `.env` и настройте:

| Переменная | Описание |
| :--- | :--- |
| `VITE_API_URL` | URL API бота |
